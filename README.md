# Hmamouch Shop

Welcome to **Hmamouch Shop**! This project is an e-commerce platform designed to provide a seamless online shopping experience.

## Features

- User authentication and registration
- Product browsing and search
- Shopping cart and checkout
- Order management
- Responsive design

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
git clone https://github.com/HMAMOUCHYOUSSEF/hmamouch-shop.git
cd hmamouch-shop
npm install
```

### Running the App

```bash
npm start
```

The app will be available at `http://localhost:3000`.
